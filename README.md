# SPAWF boilerplate with Webpack 5

## Installation
```
npm install
```

# Developement environment (front)
```
npm run dev
```

# Developement environment (back)
```
npm run build
./node_modules/nodemon/bin/nodemon.js functions/index.js
```

# Production environment
```
npm run build
```

# Run server
```
node functions/index.js
```
